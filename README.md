# Azure-Databricks
Hands-on learning session on Databricks

## Overview
This repository contains comprehensive Databricks tutorials covering:
- **Basics**: SparkSession, Magic Commands, Data Reading
- **DBFS**: Databricks File System operations and utilities
- **Delta Lake**: Delta format, write modes, managed/external tables, versioning
- **Fundas**: Fundamentals and core concepts
- **Config**: Azure authentication and configuration
- **PySpark Codes**: Common PySpark operations and transformations

## Getting Started
Each notebook is designed as a hands-on learning module. Start with Basics.ipynb and progress through the other notebooks.
